# 📣 CTA PATTERNS — Call-to-Action Library

**Purpose**: Catalog of high-converting CTAs with timing analysis
**Key Insight**: CTA placement = as important as CTA wording

---

## CTA PHILOSOPHY

### The Algorithm Truth
Instagram rewards **native engagement** (comments/shares/saves) over external clicks.

**Hierarchy of CTA Value** (for reach):
1. **Comment CTA** → Highest algo boost
2. **Save CTA** → Strong signal (rewatch intent)
3. **Share CTA** → Moderate boost
4. **Follow CTA** → Low algo value (but good for growth)
5. **Link click** → Lowest (pulls them off-platform)

**Strategy**: Front-load comment CTAs, back-load link CTAs

---

## 1. COMMENT CTAs (Highest Converting)

### Pattern A: Single-Word Command
**Formula**: `Comment '[WORD]' for [BENEFIT]`

**Examples from @divy.kairoth**:
- ✅ "Comment 'Free' for website naam" (3.09M plays)
- ✅ "Comment 'SMS' to block spam calls" (523K plays)
- ✅ "Comment 'Link' and I'll DM you"

**Why It Works**:
- Zero cognitive load (one word = easy)
- Algorithm boost (each comment = reach multiplier)
- Creates reciprocity (I give, you give)

**Conversion Rate**: 1-3% of viewers will comment

---

### Pattern B: Curiosity Unlock
**Formula**: `Comment '[WORD]' to unlock [MYSTERY]`

**Examples**:
- "Comment 'Secret' for the hidden feature"
- "Type 'Naam' and I'll reveal the app"
- "Comment 'Method' for full tutorial"

**Why It Works**:
- Information gap drives action
- "Unlock" implies exclusive access
- Creates two-step engagement (comment + wait for DM)

**Conversion Rate**: 2-4% (higher than Pattern A)

---

### Pattern C: Poll/Vote Comment
**Formula**: `Comment [OPTION A] or [OPTION B]`

**Examples**:
- "Comment 'Worth it' or 'Scam'"
- "Type 'iPhone' or 'Android'"
- "Comment 'Yes' if you agree, 'No' if not"

**Why It Works**:
- Low friction (opinion vs action)
- Creates conversation in comments
- Viewers read others' comments (engagement loop)

**Conversion Rate**: 3-5% (highest of comment CTAs)

---

### Pattern D: Emoji Response
**Formula**: `Comment [EMOJI] if [CONDITION]`

**Examples**:
- "Drop 🔥 if this helped"
- "Comment 💀 if this happened to you"
- "Type ✅ if you want part 2"

**Why It Works**:
- Emoji = even lower friction than words
- Emotional connection (emoji = feeling)
- Mobile-native (emoji keyboard always accessible)

**Conversion Rate**: 2-3%

---

### Comment CTA Timing Analysis

| Placement | Effectiveness | Use Case |
|-----------|---------------|----------|
| **0-3s** (hook) | 9/10 | Resource sharing, DM offers |
| **15-20s** (mid) | 6/10 | Reminder for slower viewers |
| **45-50s** (end) | 8/10 | Catch completers, urgency |

**Winning Pattern** (from 3.09M reel):
- Primary CTA: 0s (in hook)
- Reminder: 45s (after value delivered)
- Final push: 49s ("right now!")

**Avoid**: Repeating same CTA 5+ times (feels desperate)

## 2. FOLLOW CTAs (Growth Focused)

### Pattern A: Value-Add Follow
**Formula**: `Follow for [SPECIFIC CONTENT TYPE]`

**Examples**:
- ❌ "Follow for more" (generic, weak)
- ✅ "Follow for daily tech hacks like this"
- ✅ "Follow if you want me to expose more scams"

**Why It Works**:
- Promises specific value stream
- Self-selects interested audience
- Sets content expectation

**Conversion Rate**: 0.5-1% of viewers

---

### Pattern B: Proof-Based Follow
**Formula**: `I post [X] [VALUABLE THING] every [FREQUENCY]`

**Examples**:
- "I post 3 money-saving apps every week"
- "Every Monday I reveal a new startup idea"
- "Daily DM tricks you've never seen"

**Why It Works**:
- Quantifies value (X per Y)
- Creates expectation loop
- Implies consistency/reliability

**Conversion Rate**: 0.8-1.5%

---

### Pattern C: FOMO Follow
**Formula**: `Follow before [EVENT] or you'll miss [BENEFIT]`

**Examples**:
- "Follow before I delete this account"
- "Follow before this gets banned"
- "I'm doing a giveaway at 10K followers"

**Why It Works**:
- Time pressure creates urgency
- Fear of missing out
- Social proof (others are following)

**Conversion Rate**: 1-2% (but can feel manipulative)

---

### Follow CTA Timing Analysis

| Placement | Effectiveness | Rationale |
|-----------|---------------|-----------|
| **Pre-10s** | 3/10 | Haven't earned trust yet |
| **25-35s** | 7/10 | Value delivered, peak trust |
| **Post-50s** | 5/10 | Only completers see it |

**Winning Pattern**:
- Deliver value first (0-30s)
- Ask for follow (30-35s)
- Return to value (35-55s)
- End with comment CTA (55-60s)

## 3. SAVE CTAs (Quality Signal)

### Pattern A: Tutorial Save
**Formula**: `Save this so you can [USE CASE]`

**Examples**:
- "Save this for when you need it"
- "Save this to try later"
- "Bookmark this, you'll thank me"

**Why It Works**:
- Acknowledges they won't act now
- Implies rewatch value
- Saves = algorithm loves you

**Conversion Rate**: 2-4% of viewers

---

### Pattern B: Reference Save
**Formula**: `Save this as your [RESOURCE TYPE]`

**Examples**:
- "Save this as your app library"
- "Bookmark this as your go-to list"
- "Save this cheat sheet"

**Why It Works**:
- Positions content as valuable resource
- Creates mental categorization
- Implies you'll reference multiple times

**Conversion Rate**: 3-5%

---

### Save CTA Timing Analysis

| Placement | Effectiveness | Rationale |
|-----------|---------------|-----------|
| **Mid-video (20-30s)** | 8/10 | After value revealed |
| **End-card (55-60s)** | 7/10 | Recap value |
| **Never at start** | 1/10 | No value demonstrated yet |

**Winning Pattern**:
- Show value (0-25s)
- "Save this for later" (27s)
- Continue delivering (30-55s)
- End with comment CTA

## 4. SHARE CTAs (Viral Accelerators)

### Pattern A: Tag-a-Friend
**Formula**: `Tag someone who [CHARACTERISTIC]`

**Examples**:
- "Tag that friend who always asks you tech questions"
- "Send this to your mom (she needs this)"
- "Tag your bestie who gets scammed every week"

**Why It Works**:
- Creates social interaction
- Personalizes the ask
- Viewer thinks of specific person = higher conversion

**Conversion Rate**: 0.5-1% (low but high-value)

---

### Pattern B: Problem Solver Share
**Formula**: `Share this with anyone who [HAS PROBLEM]`

**Examples**:
- "Share this with anyone who hates spam calls"
- "Send this to your startup friends"
- "Forward this to your WhatsApp groups"

**Why It Works**:
- Positions viewer as helper
- Gives them social capital
- Widens reach organically

**Conversion Rate**: 0.3-0.8%

---

### Share CTA Timing Analysis

| Placement | Effectiveness | Rationale |
|-----------|---------------|-----------|
| **After value bomb (30s)** | 8/10 | "Wow" moment = share trigger |
| **End of video (58s)** | 6/10 | Recap share |
| **Never multiple times** | 2/10 | Feels pushy |

**Winning Pattern**:
- Deliver high-value insight (25-30s)
- "Share this with [person]" (31s)

## 5. LINK CTAs (Lowest Priority)

### Pattern A: Bio Link (Soft Push)
**Formula**: `Link in bio for [SPECIFIC THING]`

**Examples**:
- ❌ "Link in bio" (vague)
- ✅ "Full list in bio"
- ✅ "Download spreadsheet - link in bio"

**Why It Works**:
- Specific = viewer knows what they'll get
- Bio links don't hurt algo (no external exit)
- Can track clicks

**Click Rate**: 0.1-0.3% of viewers

---

### Pattern B: DM for Link
**Formula**: `Comment '[WORD]' and I'll DM you the link`

**Examples**:
- "Comment 'Link' for website access"
- "Type 'Free' and I'll send you the resource"

**Why It Works**:
- Two-step = filters serious people
- Gets comment (algo boost) PLUS lead
- Creates one-on-one connection

**Conversion Rate**: 1-2% comment → 50-70% actually DM

---

|-----------|---------------|-----------|
| **Pre-20s** | 2/10 | Too early, no trust |
| **40-50s** | 7/10 | Value delivered, earned ask |
| **End screen (58-60s)** | 8/10 | Final CTA for completers |

## 6. MULTI-CTA SEQUENCING (Advanced)

### The Waterfall Model
**Concept**: Layer CTAs by effort level (easy → hard)

**Example from 3.09M reel**:
```
0s:   "Comment 'Free'" ← Lowest friction
45s:  "Link in bio" ← Medium friction
49s:  "Comment right now!" ← Urgency repeat
```

**Conversion Logic**:
- 3% comment at 0s
- 0.2% click bio at 45s
- Additional 1% comment at 49s (reminded)
- Total: 4.2% took action (vs 3% single CTA)

---

### The Value Sandwich
**Concept**: CTA → Value → CTA

**Example**:
```
0-5s:   Hook + CTA ("Comment 'Free'")
6-50s:  Pure value delivery
51-60s: Repeat CTA + urgency ("Do it now")
```

**Why It Works**:
- Early CTA catches eager viewers
- Value justifies the ask
- End CTA catches those who needed convincing

## 8. PLATFORM-SPECIFIC CTA TRICKS

### Instagram Reels
- ✅ Comment CTAs (algo loves them)
- ✅ Save CTAs (strong ranking signal)
- ⚠️ Link CTAs (bio only, track clicks)
- ❌ External CTAs (kills reach)

### YouTube Shorts
- ✅ Subscribe CTAs (channel growth)
- ✅ Comment CTAs (engagement)
- ⚠️ Link CTAs (pinned comment)
- ❌ Too many CTAs (viewers less tolerant)

### TikTok
- ✅ Comment CTAs (duet/stitch bait)
- ✅ Follow CTAs (algo neutral)
- ⚠️ Save CTAs (less important than IG)
- ✅ Stitch/Duet CTAs (unique to platform)

## 9. A/B TESTING FRAMEWORK

### Test 1: CTA Wording
- Version A: "Comment 'Free' below"
- Version B: "Type 'Free' in comments"
- Version C: "Drop 'Free' for link"

**Winner Typically**: "Comment [X]" (standard, clear)

---

### Test 2: CTA Placement
- Version A: CTA at 0s (hook)
- Version B: CTA at 30s (mid)
- Version C: CTA at 55s (end)

**Winner Typically**: 0s for comment CTAs, 55s for link CTAs

---

### Test 3: Single vs Multi-CTA
- Version A: Only comment CTA
- Version B: Comment + Follow + Save

**Winner Typically**: Single CTA (focus = higher conversion)

## 10. ANTI-PATTERNS (CTA Fails)

| Mistake | Why It Fails | Fix |
|---------|--------------|-----|
| **"Like and follow"** | Too generic, no value prop | Specific value promise |
| **Multiple CTAs at once** | Decision paralysis | One primary CTA |
| **CTA before value** | Haven't earned it | Deliver first, ask second |
| **Vague CTA** | "Link in bio" (for what?) | "Full tutorial in bio" |
| **Desperate tone** | "Please guys please" | Confident, value-based ask |
| **External links early** | Algo penalty | DM unlock or bio only |

## 13. CTA VOICE & TONE

### Casual/Friendly (Works for @divy.kairoth)
- "Bhai, comment 'Free' neeche"
- "Dost, just type 'Link' and I'll DM you"
- "Yaar, save this for later, trust me"

### Authoritative/Expert
- "If you're serious about [X], comment below"
- "This is critical - save this immediately"
- "Don't scroll past this without commenting"

### Humorous/Relatable
- "Comment 'Help' if you relate 💀"
- "Drop 🔥 if this made you feel personally attacked"
- "Tag that friend who needs to see this (you know who)"

**Rule**: Match tone to content and audience. Tech tips = casual works.

```

├─ Engagement (reach) → Comment CTA (0-3s placement)

├─ Growth → Follow CTA (30-35s placement)

└─ Conversions → DM unlock CTA (45-50s placement)
```

**Golden Rule**: One primary CTA per video. Everything else = secondary.
